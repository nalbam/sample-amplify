# sample-amplify

## Install and configure the Amplify CLI.

```bash
npm install -g @aws-amplify/cli
```

## amplify

```
amplify init

amplify auth add
amplify analytics add

amplify push
amplify publish

```

## rewrite

```
</^[^.]+$|\.(?!(css|gif|ico|jpg|js|png|txt|svg|woff|ttf)$)([^.]+$)/>    /index.html    200
```
